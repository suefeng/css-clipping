CSS Clipping Demo

Here's a simple clip-path demo. First there is a a rectangular image. Then when applying a clip-path style on the image, the clip-path cuts away parts of the rectangular image to form a shape. In this case, it's a hexagon. The basic clip-path code is this: clip-path: polygon(25% 5%, 75% 5%, 100% 50%, 75% 95%, 25% 95%, 0% 50%); As you see, there are 6 coordinates. The coordinates are x and y values from top left to all the points in a clockwise direction.

Here's an example of a star. It has 10 coordinates. The clip-path code looks like this: clip-path: polygon(50% 0%, 63% 38%, 100% 38%, 69% 59%, 82% 100%, 50% 75%, 18% 100%, 31% 59%, 0% 38%, 37% 38%);

Here's an example of a cutout that is hollow in the middle. This one has a css gradient.

Putting together a composition using just one image, you can see how useful css clip-path can be in making something less boxy.
